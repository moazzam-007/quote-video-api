# Background Music / SFX Folder

Aap apne sabhi background sounds (Nature sounds, Lofi beats, etc.) yahan par paste kar sakte hain.

- **Formats allowed:** `.mp3`, `.wav`, `.m4a`
- System inmein se kisi ek ko randomly chunn kar automatically video ke background mein lagayega.
- Iski volume khud-ba-khud kam ho jayegi taaki main voice (TTS) clearly sunayi de.
